Easily convert Rank Math FAQ blocks into an interactive accordion format.

== Description ==

Convert Rank Math FAQ to Accordion is a simple yet powerful plugin that transforms your Rank Math FAQ blocks into an interactive accordion. This enhancement improves user experience by making your FAQs more engaging and easier to navigate.

== ⚡ Features ==

* Automatically converts Rank Math FAQ blocks to accordion
* Customizable colors for questions and answers
* Adjustable font sizes
* Customizable border style and border radius
* Option to enable/disable counter for FAQ items and initial expand first FAQ
* Easy to use plugin settings page in WordPress admin
* Responsive design for optimal viewing on all devices

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/convert-rank-math-faq-to-accordion` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> Rank Math FAQ Accordion screen to configure the plugin

== Frequently Asked Questions ==

= Does this plugin work with other FAQ plugins? =

No, this plugin is specifically designed to work with Rank Math FAQ blocks.

= Can I customize the appearance of the accordion? =

Yes, you can customize colors, font sizes, and border styles through the settings page.
