# Inmopress Frontend Plugin

Plugin que proporciona el panel frontend para Inmopress con shortcodes para listados y formularios.

## Shortcodes Disponibles

- `[inmopress_dashboard]` - Dashboard principal con estadísticas
- `[inmopress_inmuebles_list]` - Listado de inmuebles
- `[inmopress_inmueble_form]` - Formulario crear/editar inmueble
- `[inmopress_clientes_list]` - Listado de clientes
- `[inmopress_cliente_form]` - Formulario crear/editar cliente
- `[inmopress_visitas_list]` - Listado de visitas
- `[inmopress_visita_form]` - Formulario crear/editar visita
- `[inmopress_propietarios_list]` - Listado de propietarios
- `[inmopress_events_list]` - Listado de eventos
- `[inmopress_event_form]` - Formulario crear/editar evento
- `[inmopress_today_tasks]` - Tareas de hoy (dashboard)

## Uso

Crear páginas en WordPress y añadir los shortcodes correspondientes. Las páginas deben estar configuradas con template Full Width en Astra.
